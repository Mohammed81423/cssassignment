universal selector       element selector       id selector       class selector      

universal selector

The * selector selects all elements. The * selector can also select all elements inside another element

element selector

The element element selector in CSS is used to style only those elements which are the children of the specific parent.

id selector

A CSS ID selector uses the ID attribute of an HTML element to select one unique element on a page

class selector

The class selector selects HTML elements with a specific class attribute. To select elements with a specific class, write a period (.) character, followed by the class name.
